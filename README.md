# php-calculator-2-with-history-log
This  calculator has 5 buttons +-*/2 and clear with history log

The faster way to get 2018 starting form 0 is:
0 + 2 = 2 
2 * 2 = 4 
4 * 2 = 8 
8 * 2 = 16 
16 * 2 = 32 
32 * 2 = 64 
64 * 2 = 128 
128 - 2 = 126 
126 * 2 = 252 
252 * 2 = 504 
504 * 2 = 1008 
1008 * 2 = 2016 
2016 + 2 = 2018 

13 buttons to press
